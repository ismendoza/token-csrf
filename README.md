# token-csrf
Generar tokens de 64 caracteres
## Instalación
```
$ npm i token-csrf
```
## Implementación

### Módulo estandar de javascript
```
const getToken = require('token-csrf')
```
### Módulo ECMAScript
```
import getToken from 'token-csrf'
```
### Utilización
```
const token = getToken()
```
